# 06 — Instalación Box86 y Box64

Preparación del entorno de compatibilidad x86 y x86_64 antes de instalar Wine ARM.

Sistema:

- Debian dentro de Proot
- Android + Termux (HOST)

---

## 1 — Habilitar arquitectura ARMHF (Box86)

Box86 requiere librerías ARM de 32 bits.

Agregar soporte:

sudo dpkg --add-architecture armhf

Actualizar repositorios:

sudo apt update

Instalar librerías necesarias:

sudo apt install \
libc6:armhf \
libstdc++6:armhf \
libgcc-s1:armhf

---

## 2 — Instalar Box64

Agregar repositorio:

sudo wget https://ryanfortner.github.io/box64-debs/box64.list \
-O /etc/apt/sources.list.d/box64.list

wget -qO- https://ryanfortner.github.io/box64-debs/KEY.gpg \
| sudo gpg --dearmor \
-o /etc/apt/trusted.gpg.d/box64-debs-archive-keyring.gpg

Actualizar:

sudo apt update

Instalar:

sudo apt install box64-android

Verificar:

box64 --version

---

## 3 — Instalar Box86

Agregar repositorio:

sudo wget https://ryanfortner.github.io/box86-debs/box86.list \
-O /etc/apt/sources.list.d/box86.list

wget -qO- https://ryanfortner.github.io/box86-debs/KEY.gpg \
| sudo gpg --dearmor \
-o /etc/apt/trusted.gpg.d/box86-debs-archive-keyring.gpg

Actualizar:

sudo apt update

Instalar:

sudo apt install box86-android

Verificar:

box86 --version

---

## 4 — Problema común

Error:

cannot execute: required file not found

Causa:

Faltan librerías ARMHF necesarias.

Solución:

sudo dpkg --add-architecture armhf

sudo apt update

sudo apt install libc6:armhf libstdc++6:armhf libgcc-s1:armhf

---

## 5 — Validación Box86

file /usr/local/bin/box86

Debe mostrar:

ELF 32-bit ARM executable

---

## 6 — Validación Box64

box64 --version

Debe iniciar correctamente.

---

## 7 — Notas importantes

Mensajes debconf relacionados con Dialog son normales en Proot y pueden ignorarse.

No afectan la instalación.
